# Scalable Newsletter App

A full-stack application for managing newsletters and events.

## Tech Stack
- **Backend:** Node.js, Express, MongoDB (Mongoose ORM)
- **Frontend:** React, Vite, TailwindCSS
- **Authentication:** JWT, bcrypt
- **Database:** MongoDB Atlas

## Setup

1.  **Clone the repository.**
2.  **Backend:**
    ```bash
    cd backend
    npm install
    cp ../.env.example .env
    npm run dev
    ```
3.  **Frontend:**
    ```bash
    cd frontend
    npm install
    npm run dev
    ```

## Features (MVP)
- Admin Dashboard
- Create/Edit Events
- Compose Newsletters
- Public Landing Page
- Subscription Form
